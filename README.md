# WorthWise (Capstone Project)

**WorthWise** is a financial decision learning platform for teens and young adults.  
The project focuses on building practical money judgment through simulation instead of theory-only learning.

## Project Goal
Help users practice real-world financial choices (saving, investing, handling shocks, and long-term planning) in a safe, guided environment.

## Core Modules (Project Scope)
- Financial profile and foundation setup
- SIP/portfolio growth simulation with uncertainty
- Decision timeline with consequence-based outcomes
- Learning levels (Explorer, Analyst, Strategist)
- Reflection layer that explains the "why" behind decisions
